# Configuration GitHub

## Étape 1: Créer Repository GitHub
1. Aller sur https://github.com/new
2. Nom: livraison-ma
3. Visibilité: Private (recommandé)
4. Ne pas initialiser avec README (on a déjà le code)
5. Cliquer "Create repository"

## Étape 2: Connecter Local → GitHub
git remote add origin https://github.com/VOTRE-USERNAME/livraison-ma.git
git branch -M main
git push -u origin main

## Étape 3: Sauvegardes Futures
# Après chaque modification:
git add .
git commit -m "Description"
git push

# Tout est sauvegardé dans le cloud automatiquement!

## Restauration (si PC crashe)
Sur nouveau PC:
git clone https://github.com/VOTRE-USERNAME/livraison-ma.git
cd livraison-ma
npm install
# Code récupéré!
